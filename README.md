# Text Summarization Web App

This is a simple web application for text summarization. It takes input text and generates a summary using natural language processing techniques.

## Usage

1. Navigate to https://text-summarization12.streamlit.app/
2. Paste or type the text you want to summarize into the input box.
3. Also you can upload the text document
4. Click on the "Summarize" button.
5. The summarized text will be displayed below.

## Features

- Summarizes input text quickly and accurately.
- Easy-to-use interface.

## About

This web app is built using Streamlit, a Python library for creating interactive web applications. It utilizes natural language processing models for text summarization.
